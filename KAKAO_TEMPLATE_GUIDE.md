# 카카오톡 알림톡 템플릿 등록 가이드

## 1. 카카오톡 비즈니스 센터 접속

1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속
2. 카카오톡 비즈니스 계정으로 로그인
3. 대시보드 접속

## 2. 플러스친구 등록 (필수)

### 2.1 플러스친구 생성
1. 비즈니스 센터 → 채널 → 플러스친구 만들기
2. 플러스친구 정보 입력:
   - 이름 (예: "보듬교육")
   - 프로필 사진
   - 소개 메시지
3. 검수 신청 및 승인 대기 (일반적으로 1-2일 소요)

### 2.2 플러스친구 ID 확인
1. 승인 완료 후 플러스친구 관리 페이지 접속
2. 플러스친구 ID 확인 (예: "@보듬교육")
3. 이 ID를 솔라피 연동 시 사용합니다

## 3. 알림톡 템플릿 등록

### 3.1 템플릿 만들기

**메뉴 찾는 방법**:
1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속 및 로그인
2. **플러스친구 선택**: 관리 중인 플러스친구 목록에서 해당 플러스친구를 클릭
3. **비즈메시지 메뉴**: 플러스친구 관리 페이지에서 좌측 또는 상단 메뉴에서 **"비즈메시지"** 클릭
4. **알림톡 메뉴**: "비즈메시지" 하위 메뉴에서 **"알림톡"** 클릭
5. **템플릿 만들기** 버튼 클릭

**템플릿 작성**:
1. 템플릿 유형 선택: **알림톡**
2. 템플릿 정보 입력 (이름, 카테고리 등)
3. 메시지 내용 작성 (변수 포함)

### 3.2 템플릿 내용 작성

#### 기본 정보
- **템플릿 이름**: "과제 진행 상황 안내" 또는 "클리닉 대장 안내" (자유롭게 설정 가능)
- **카테고리**: 일반/결제/예약 등 (선택)
- **버튼**: 선택사항 (링크 버튼 추가 가능)

#### 메시지 내용 예시 (과제 진행 상황)

```
[과제 진행 상황 안내]

${title}

${content}

자세한 내용은 학원에서 확인해주세요.
```

**중요**: 변수는 `${변수명}` 형식으로 작성해야 합니다. (카카오톡 매뉴얼 기준)

### 3.3 템플릿 변수 설정

템플릿 작성 화면에서 변수를 추가합니다:

1. 텍스트 입력란에 직접 `${변수명}` 형식으로 입력 (예: `${title}`, `${content}`)
2. 변수 이름 규칙: 영문 대소문자, 숫자, 언더바(_)만 사용 가능

#### 권장 변수 목록

**과제 진행 상황용**:

| 변수명 | 설명 | 예시 |
|--------|------|------|
| `title` | 제목 (학교/학년/반) | "과천중앙고등학교 2학년 월요일반 과제 진행상황" |
| `content` | 과제 상황 내용 | 학생별 진행 상황 텍스트 |

**클리닉 대장용**:

| 변수명 | 설명 | 예시 |
|--------|------|------|
| `week` | 주차 정보 | "2024년 45주차 (11/4~)" |
| `content` | 클리닉 대장 전체 내용 | 요일별, 시간별로 정렬된 학생 목록 (상세 포맷은 6.2절 참고) |

### 3.4 템플릿 제한사항 확인

- **최대 길이**: 메시지 본문 1000자 이내
- **변수 개수**: 최대 10개
- **이미지**: 알림톡은 이미지 사용 불가 (친구톡만 가능)

## 4. 템플릿 검수 신청 (중요!)

⚠️ **템플릿을 저장했다고 끝이 아닙니다! 반드시 검수 신청을 해야 합니다.**

### 4.1 검수 신청 방법

**카카오톡 개발자 콘솔 (kakao developers)에서:**

1. 템플릿 목록 페이지에서 **템플릿 제목을 클릭**하거나, **각 행의 세로 점 3개(⋮) 메뉴**를 클릭
2. 템플릿 상세 페이지로 이동
3. 템플릿 상세 페이지에서 **"검수 신청"** 또는 **"승인 요청"** 버튼을 찾아 클릭
   - 버튼 위치: 상단 액션 영역 또는 하단 저장 버튼 근처
   - 템플릿 상태가 "임시저장" 또는 "대기"인 경우 표시됨

**또는 카카오톡 비즈니스 센터에서:**

1. 카카오톡 비즈니스 센터 접속
2. 플러스친구 선택 → **비즈메시지 → 알림톡** 메뉴로 이동
3. **템플릿 목록** 또는 **템플릿 관리** 페이지에서 작성한 템플릿을 찾아서 클릭
4. 템플릿 상세 페이지에서 **"검수 신청"** 또는 **"승인 요청"** 버튼 클릭
   - 버튼이 보이지 않으면: 템플릿 상태가 "임시저장" 또는 "대기"인지 확인
   - 이미 검수 신청한 경우: 상태가 "검수 중" 또는 "검수 대기"로 변경됨
5. 검수 신청 사유 작성 (선택사항)
6. 검수 승인 대기 (일반적으로 1-3일 소요)

### 4.2 템플릿 상태 확인

템플릿 목록에서 상태를 확인할 수 있습니다:
- **임시저장**: 저장만 하고 검수 신청 안 함 → **검수 신청 필요!**
- **검수 대기**: 검수 신청 완료, 승인 대기 중
- **검수 중**: 카카오에서 검수 진행 중
- **승인**: 검수 완료, 사용 가능 ✅
- **반려**: 검수 거부, 수정 후 재신청 필요

### 검수 승인 기준
- 카카오톡 정책 준수
- 스팸성 메시지 아님
- 변수 사용 적절
- 문구가 명확하고 이해하기 쉬움

## 5. 솔라피에 템플릿 연동

### 5.1 솔라피 대시보드 접속
1. [솔라피](https://solapi.com) 로그인
2. 대시보드 → **메시지** → **알림톡** (또는 **카카오톡**) → **템플릿 관리** 메뉴로 이동

### 5.2 카카오톡 비즈니스 채널 연동
1. 솔라피 → 채널 관리 → 카카오톡 채널 추가
2. 카카오톡 비즈니스 계정 연동
3. 플러스친구 선택 및 연결
4. 연동 완료 후 승인된 템플릿 자동 동기화

### 5.3 템플릿 코드 확인
1. 솔라피 → 알림톡 → 템플릿 목록
2. 승인된 템플릿 목록 확인
3. **템플릿 코드** 복사 (예: `TL-1234567890`)

이 템플릿 코드를 환경 변수 `SOLAPI_TEMPLATE_CODE`에 설정합니다.

## 6. 템플릿 예시

### 6.1 과제 진행 상황용 템플릿

#### 예시 1: 간단한 템플릿

```
[과제 진행 상황]

${title}

${content}

보듬교육
```

**변수**:
- `${title}`: 과천중앙고등학교 2학년 월요일반 과제 진행상황
- `${content}`: 학생별 과제 진행 상황 (체크된 항목만 표시)

#### 예시 2: 상세한 템플릿

```
📋 ${title}

━━━━━━━━━━━━━━━━━━━━

${content}

━━━━━━━━━━━━━━━━━━━━

자세한 내용은 학원에서 확인해주세요.
```

**변수**:
- `${title}`: 제목
- `${content}`: 과제 진행 상황 상세 내용

### 6.2 클리닉 대장용 템플릿

클리닉 대장의 모든 정보를 포함하는 템플릿입니다.

#### 템플릿 내용

```
📋 클리닉 대장 안내

${week}

━━━━━━━━━━━━━━━━━━━━

${content}

━━━━━━━━━━━━━━━━━━━━

보듬교육
```

**변수 설명**:
- `${week}`: 주차 정보 (예: "2024년 45주차 (11/4~)")
- `${content}`: 클리닉 대장 전체 내용 (요일별, 시간별로 정렬된 학생 목록)

#### `${content}` 변수 포맷 예시

```
📅 월요일
━━━━━━━━━━━━━━━━━━━━

👤 홍길동 (2학년천) - 화목반
⏰ 시간: 17시
✓ 출결: O
👨‍🏫 담당 조교: 김선생
🚪 등원: 16:50 / 하원: 19:30
📱 문자 완료: O
✅ 시험 확인: 완료
📝 비고: 수학 문제 해결

👤 이영희 (1학년앙) - 월금반
⏰ 시간: 18시
✓ 출결: O
👨‍🏫 담당 조교: 박선생
🚪 등원: 17:45 / 하원: 20:00

📅 화요일
━━━━━━━━━━━━━━━━━━━━

👤 김철수 (3학년천) - 금요일반
⏰ 시간: 17시
✓ 출결: O
...
```

#### 템플릿 등록 시 주의사항

1. **변수 이름 정확히 입력**: `${week}`, `${content}` (대소문자 구분)
2. **템플릿 길이**: `${content}` 변수에 많은 내용이 들어갈 수 있으므로, 템플릿 자체는 짧게 작성
3. **검수 시 주의**: 실제 변수 값을 포함한 샘플 메시지를 작성하여 검수 신청

#### 템플릿 검수 신청 시 샘플 값

**변수 값 예시**:
```json
{
  "week": "2024년 45주차 (11/4~)",
  "content": "📅 월요일\n━━━━━━━━━━━━━━━━━━━━\n\n👤 홍길동 (2학년천) - 화목반\n⏰ 시간: 17시\n✓ 출결: O\n👨‍🏫 담당 조교: 김선생\n🚪 등원: 16:50 / 하원: 19:30\n\n..."
}
```

### 6.3 학생별 개별 전송용 (변수 3개 사용)

```
[과제 진행 상황 안내]

${student_name} 학생의 과제 진행 상황입니다.

${title}

${content}

자세한 내용은 학원에서 확인해주세요.
```

**변수**:
- `${student_name}`: 학생 이름
- `${title}`: 제목
- `${content}`: 해당 학생의 과제 진행 상황

## 7. 템플릿 내용 생성 로직

프론트엔드에서 템플릿 변수에 맞게 내용을 생성해야 합니다:

```javascript
// 예시: content 변수 생성
let content = '';
students.forEach(student => {
  content += `\n👤 ${student}\n`;
  // 완료한 과제만 표시
  if (완료된_과제_있음) {
    content += `✅ ${완료된_과제_목록}\n`;
  }
});

const variables = {
  title: getTitle(), // "과천중앙고등학교 2학년 월요일반 과제 진행상황"
  content: content // 학생별 진행 상황
};
```

## 8. 템플릿 수정 및 재검수

1. 템플릿 수정이 필요한 경우:
   - 비즈니스 센터 → 알림톡 → 템플릿 관리
   - 수정할 템플릿 선택 → 수정
   - 내용 변경 후 **재검수 신청**

2. 재검수 승인 대기 (1-3일)

## 9. 주의사항

1. **템플릿 검수 시간**: 처음 등록 시 1-3일 소요
2. **변수명 통일**: 템플릿의 변수명과 코드의 변수명이 정확히 일치해야 함
3. **특수문자**: 이모지 사용 가능하지만 과도한 사용 시 검수 거부 가능
4. **마케팅 문구 금지**: 알림톡은 마케팅 목적 사용 불가
5. **길이 제한**: 1000자 이내로 작성

## 10. 문제 해결

### 템플릿 검수 거부 시
- 카카오톡 정책 위반 사항 확인
- 마케팅성 문구 제거
- 문구 수정 후 재신청

### 템플릿이 솔라피에 동기화되지 않을 때
- 카카오톡 비즈니스 채널 연동 상태 확인
- 플러스친구 승인 상태 확인
- 솔라피 고객센터 문의
