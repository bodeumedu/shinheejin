📱 카카오톡 알림톡 템플릿 테스트

💡 사용 방법: 변수를 입력한 후 미리보기를 확인하고, 실제 발송 테스트를 진행하세요. 검수 승인 전에는 발송이 실패할 수 있습니다.
📌 중요: 이 파일은 웹 서버를 통해 실행하거나, Vercel 배포 사이트에서 사용하세요. 로컬 파일로 직접 열면 API 호출이 실패할 수 있습니다.

템플릿 선택

1. 변수 값 입력 (과제 진행 상황)

1. 변수 값 입력 (클리닉 대장)

2. 템플릿 미리보기

3. 실제 발송 테스트

⚠️ 주의: 템플릿 코드는 카카오톡 비즈니스 센터에서 검수 승인 후 솔라피에서 확인할 수 있습니다. 검수 전에는 템플릿 코드가 없어 발송 테스트가 불가능할 수 있습니다.
💡 비워두면 현재 페이지의 도메인에서 API를 호출합니다. 다른 서버를 사용하려면 전체 URL을 입력하세요. 💡 템플릿 코드 확인 방법: 솔라피 → 메시지 → 알림톡 → 템플릿 목록 → 템플릿 코드 복사
