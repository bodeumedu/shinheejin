# 검수 신청 버튼을 찾을 수 없을 때 빠른 해결책

## 상황
- 카카오톡 개발자 콘솔에서 템플릿 상세 페이지를 봤는데
- "저장" 버튼만 보이고 "검수 신청" 버튼이 없음

## 해결책: 카카오톡 비즈니스 센터 사용 (가장 확실한 방법)

카카오톡 개발자 콘솔과 비즈니스 센터는 다른 시스템입니다. 알림톡 템플릿 검수는 **비즈니스 센터**에서 해야 합니다.

### 단계별 안내

1. **카카오톡 비즈니스 센터 접속**
   - https://business.kakao.com
   - 플러스친구 계정으로 로그인

2. **플러스친구 선택**
   - "보듬교육_과제봇" 또는 해당 플러스친구 클릭

3. **알림톡 메뉴 찾기**
   - 좌측 메뉴에서 **"비즈메시지"** 클릭
   - 또는 상단 메뉴에서 **"알림톡/친구톡"** 클릭

4. **템플릿 만들기**
   - **"템플릿 만들기"** 또는 **"+ 템플릿 추가"** 버튼 클릭
   - 템플릿 내용 작성 (변수 포함)
   - **"저장"** 클릭

5. **검수 신청**
   - 템플릿 목록으로 돌아가기
   - 저장한 템플릿 클릭
   - **"검수 신청"** 또는 **"승인 요청"** 버튼 클릭
   - 이 버튼은 여기서 확실히 보입니다!

## 또는: 개발자 콘솔에서 저장만 하고 확인

1. 개발자 콘솔에서 템플릿 **"저장"** 클릭
2. 템플릿 목록으로 돌아가기
3. 템플릿 상태 확인:
   - "검수 대기" 또는 "검수 중"으로 변경되었다면 → 자동 검수 신청됨 ✅
   - "임시저장"으로 남아있다면 → 비즈니스 센터에서 검수 신청 필요

## 두 가지 방법 비교

| 항목 | 개발자 콘솔 | 비즈니스 센터 |
|------|-------------|---------------|
| 용도 | API 테스트/개발 | 실제 비즈니스 운영 |
| 템플릿 작성 | 가능 | 가능 |
| 검수 신청 | 버튼이 보이지 않을 수 있음 | 확실히 있음 ✅ |
| 권장 | API 테스트용 | 실제 운영용 ✅ |

**가장 확실한 방법**: 카카오톡 비즈니스 센터에서 템플릿을 만들고 검수 신청하세요!
