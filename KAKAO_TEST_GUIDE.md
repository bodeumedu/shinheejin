# 카카오톡 알림톡 전송 테스트 가이드

## 테스트 방법

### 방법 1: 테스트 HTML 페이지 사용 (권장)

1. **파일 열기**
   - `test-kakao-template.html` 파일을 브라우저에서 직접 열기
   - 또는 배포된 사이트에서 접속: `https://your-domain.vercel.app/test-kakao-template.html`

2. **템플릿 선택**
   - "과제 진행 상황" 또는 "클리닉 대장" 선택

3. **변수 값 입력**
   - 템플릿에 맞는 변수 값 입력
   - 예시 값이 이미 입력되어 있음

4. **미리보기 확인**
   - "미리보기 업데이트" 버튼 클릭
   - 실제 카카오톡 메시지 형태 확인

5. **실제 발송 테스트**
   - 전화번호 입력
   - 템플릿 코드 입력 (검수 승인 후 솔라피에서 확인)
   - "테스트 발송" 버튼 클릭

### 방법 2: 실제 페이지에서 테스트

#### 과제 진행 상황 테스트

1. **과제 진행 상황 페이지 접속**
   - 메인 메뉴 → "과제 관리 대시보드"
   - 학교/학년/반 선택 → "명단 보기"

2. **데이터 입력**
   - 학생 추가
   - 과제 체크박스 선택
   - 점수 입력

3. **카카오톡 전송**
   - "카카오톡 전송" 버튼 클릭 (탭 내에 있음)
   - 전화번호 입력
   - 템플릿 코드 입력
   - 발송

#### 클리닉 대장 테스트

1. **클리닉 대장 페이지 접속**
   - 메인 메뉴 → "🗂️ 클리닉 대장"

2. **데이터 입력**
   - 학생 불러오기 또는 수동 추가
   - 클리닉 시간, 출결 등 입력

3. **카카오톡 전송**
   - "📱 카카오톡 전송" 버튼 클릭
   - 전화번호 입력
   - 템플릿 코드 입력
   - 발송

## 템플릿 코드 확인 방법

1. **카카오톡 비즈니스 센터에서 템플릿 검수 승인 대기**
   - 템플릿이 "승인" 상태가 되어야 함

2. **솔라피에서 템플릿 코드 확인**
   - [솔라피 대시보드](https://solapi.com) 접속
   - 메시지 → 알림톡 → 템플릿 목록
   - 승인된 템플릿의 템플릿 코드 복사 (예: `TL-1234567890`)

## 테스트 체크리스트

- [ ] 템플릿이 카카오톡 비즈니스 센터에서 승인됨
- [ ] 솔라피에서 템플릿 코드 확인
- [ ] 솔라피 API 키가 Vercel 환경 변수에 설정됨
- [ ] 플러스친구 ID가 환경 변수에 설정됨
- [ ] 테스트용 전화번호 준비
- [ ] 미리보기로 메시지 형식 확인
- [ ] 실제 발송 테스트

## 문제 해결

### 발송 실패 시
- 템플릿 코드 확인
- 환경 변수 설정 확인 (Vercel 대시보드)
- 솔라피 발송 내역 확인
- API 키 및 플러스친구 ID 확인

### 템플릿 코드가 없을 때
- 카카오톡 비즈니스 센터에서 검수 승인 대기
- 검수 승인 후 솔라피에서 확인
- 검수 전에는 미리보기만 가능

## 빠른 테스트

1. `test-kakao-template.html` 파일 열기
2. 예시 데이터로 미리보기 확인
3. 템플릿 승인 후 템플릿 코드 입력
4. 본인 전화번호로 테스트 발송
