# 카카오톡 개발자 콘솔에서 검수 신청하는 방법

## 현재 페이지: 메시지 템플릿 검색 (Template Search)

텍스트가 보이는 페이지는 **"메시지 템플릿 검색"** 페이지입니다. 이 페이지에서는 검수 신청 버튼이 보이지 않습니다.

## 검수 신청 버튼 찾는 방법

### 방법 1: 템플릿 제목 클릭 (권장)

1. **템플릿 목록에서 템플릿 제목 클릭**
   - 예: "2학기 기말고사 과제 수행도 REPORT" 클릭
   - 또는: "보듬교육 클리닉 수행 REPORT" 클릭

2. **템플릿 상세 페이지로 이동**
   - 템플릿의 전체 내용이 보이는 페이지로 이동

3. **검수 신청 버튼 찾기**
   - 상단 또는 하단의 액션 버튼 영역에서 찾기
   - 버튼 이름: **"검수 신청"**, **"승인 요청"**, **"제출"** 등

### 방법 2: 세로 점 3개 메뉴 클릭

1. **각 템플릿 행의 오른쪽 끝에 있는 세로 점 3개(⋮) 아이콘 클릭**
2. **드롭다운 메뉴에서 "수정" 또는 "상세보기" 선택**
3. **템플릿 상세 페이지에서 검수 신청 버튼 찾기**

### 방법 3: 템플릿 상태 확인

템플릿 목록에서 **상태 컬럼**이 있다면 확인:
- **"임시저장"** 또는 **"대기"**: 검수 신청 가능
- **"검수 대기"** 또는 **"검수 중"**: 이미 검수 신청 완료
- **"승인"**: 검수 완료, 사용 가능

## 템플릿 상세 페이지에서 확인할 항목

템플릿 상세 페이지로 들어가면:

1. **템플릿 내용 편집 영역**
2. **변수 설정 확인** (${title}, ${content} 등)
3. **상태 표시** (임시저장, 검수 대기 등)
4. **액션 버튼들**:
   - 저장
   - **검수 신청** ← 이 버튼을 찾으세요!
   - 취소
   - 삭제

## 검수 신청 버튼이 보이지 않는 경우

### 1. 이미 검수 신청한 경우
- 상태가 "검수 대기" 또는 "검수 중"으로 표시됨
- 검수 승인을 기다리는 중이므로 버튼이 없을 수 있음

### 2. 템플릿이 이미 승인된 경우
- 상태가 "승인"으로 표시됨
- 이미 사용 가능한 상태

### 3. 권한 문제
- 템플릿 작성자가 아닌 경우
- 관리자 권한이 없는 경우

## 대안: 카카오톡 비즈니스 센터에서 검수 신청

카카오톡 개발자 콘솔에서 찾기 어려우면:

1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속
2. 플러스친구 선택
3. **비즈메시지 → 알림톡** 메뉴
4. 템플릿 목록에서 해당 템플릿 찾기
5. 템플릿 클릭 후 검수 신청

## 저장 버튼만 보이는 경우

**카카오톡 개발자 콘솔(kakao developers)은 개발자용 도구**이고, **실제 알림톡 템플릿 검수는 카카오톡 비즈니스 센터에서 해야 할 수 있습니다.**

### 해결 방법

#### 방법 1: 카카오톡 비즈니스 센터에서 확인 (권장)

1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속
2. 플러스친구 선택 (보듬교육_과제봇)
3. **비즈메시지 → 알림톡** 메뉴로 이동
4. 템플릿 목록에서 해당 템플릿 찾기
5. 템플릿 클릭 후 **검수 신청** 버튼 확인

#### 방법 2: 저장 후 상태 확인

1. 템플릿 상세 페이지에서 **"저장"** 버튼 클릭
2. 저장 완료 후 페이지를 새로고침하거나 템플릿 목록으로 돌아가기
3. 템플릿 상태가 "검수 대기" 또는 "검수 중"으로 변경되었는지 확인
4. 상태가 변경되었다면 자동으로 검수 신청된 것

#### 방법 3: 다른 위치 확인

- 템플릿 목록 페이지에서 템플릿 행을 클릭했을 때 나타나는 메뉴
- 페이지 하단의 추가 액션 버튼들
- 우측 상단의 더보기 메뉴 (⋮)

## 카카오톡 개발자 vs 비즈니스 센터 차이

- **카카오톡 개발자 콘솔 (kakao developers)**: 
  - 개발자용 API 테스트 도구
  - 메시지 템플릿 작성 및 관리
  - 검수 신청 기능이 제한적일 수 있음

- **카카오톡 비즈니스 센터 (business.kakao.com)**:
  - 실제 비즈니스 관리 도구
  - 알림톡/친구톡 템플릿 검수 신청
  - 플러스친구 관리

## 권장 사항

1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속
2. 플러스친구 선택

이렇게 하면 검수 신청 버튼을 확실히 찾을 수 있습니다.
