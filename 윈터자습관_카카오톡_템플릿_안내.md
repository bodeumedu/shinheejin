# 윈터자습관 카카오톡 템플릿 등록 가이드

## 템플릿 내용 (검수용)

```
보듬교육 윈터자습관을 든든하게 책임지겠습니다.

안녕하세요, #{학생명} 학생의 학부모님께.

보듬교육에서 #{날짜} 학습 상세 사항을 알려드립니다.

━━━━━━━━━━━━━━━━━━━━

#{스터디플래너}

━━━━━━━━━━━━━━━━━━━━

학생의 꾸준한 학습을 위해 최선을 다하겠습니다.

감사합니다.
보듬교육
```

## 템플릿 설명 (검수 신청 시 작성)

**템플릿 용도**: 윈터자습관 등록 학생의 일일 학습 계획을 학부모님께 안내하기 위한 알림톡 템플릿입니다.

**사용 목적**: 
- 학생의 일일 학습 계획을 학부모님께 정기적으로 안내
- 학습 진행 상황 및 완료도 공유
- 윈터자습관 학습 관리 및 소통

**변수 설명**:
- `#{학생명}`: 해당 학생의 이름
- `#{날짜}`: 학습 계획 날짜 (예: 1월 13일)
- `#{스터디플래너}`: 시간대별 학습 계획 상세 내용

**템플릿 특징**:
- 교육 기관의 학습 안내 목적
- 학생 개별 맞춤 학습 계획 제공
- 학부모-교육기관 간 소통 강화

## 템플릿 변수

템플릿에서 사용하는 변수는 다음과 같습니다:

1. **#{학생명}** - 학생 이름 (예: 홍길동)
2. **#{날짜}** - 날짜 (예: 1월 13일)
3. **#{스터디플래너}** - 스터디 플래너 내용 (시간대별 학습 계획)

## 템플릿 등록 시 주의사항

1. **템플릿 제목**: "윈터자습관 학습 안내" 또는 "윈터자습관 일일 학습 계획 안내"
2. **템플릿 카테고리**: 학습/교육 관련
3. **변수명 정확히 입력**: 
   - `학생명` (한글 그대로)
   - `날짜` (한글 그대로)
   - `스터디플래너` (한글 그대로)
4. **템플릿 길이**: 1000자 이내 권장
5. **이모지 사용**: ✅ 가능 (❄️, ✅ 등)

## 솔라피 템플릿 등록 절차

1. 솔라피 콘솔 접속
2. 카카오톡 → 알림톡 → 템플릿 관리
3. 템플릿 등록 클릭
4. 위의 템플릿 내용 복사하여 붙여넣기
5. 변수명 확인: `#{학생명}`, `#{날짜}`, `#{스터디플래너}`
6. 템플릿 제목 입력
7. 검수 신청
8. 검수 승인 후 템플릿 코드 확인

## 템플릿 코드 입력 방법

검수 승인 후 받은 템플릿 코드를 윈터스쿨 관리 페이지의 미리보기 모달에서 입력하면 됩니다.

템플릿 코드는 자동으로 로컬 스토리지에 저장되어 다음에도 사용할 수 있습니다.
