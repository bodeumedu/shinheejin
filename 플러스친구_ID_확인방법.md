# 플러스친구 ID 확인 방법

플러스친구 ID는 카카오톡 알림톡 발송에 필요한 중요한 정보입니다. 다음 두 가지 방법으로 확인할 수 있습니다.

## 방법 1: 솔라피 대시보드에서 확인 (가장 쉬움) ⭐

### 단계별 가이드

1. **솔라피 대시보드 접속**
   - [https://solapi.com](https://solapi.com) 접속
   - 로그인

2. **카카오톡 메뉴로 이동**
   - 좌측 메뉴에서 **"카카오톡"** 클릭
   - 또는 상단 메뉴에서 **"서비스"** → **"카카오톡"** 클릭

3. **채널 관리 페이지 열기**
   - **"채널 관리"** 또는 **"채널 목록"** 메뉴 클릭

4. **플러스친구 ID 확인**
   - 연동된 카카오톡 비즈니스 채널이 표시됩니다
   - 채널 정보에 **"플러스친구 ID"** 또는 **"PF ID"**가 표시됩니다
   - 형식: `@보듬교육` 또는 `@채널이름` 형태

5. **복사**
   - 플러스친구 ID를 복사하여 Vercel 환경 변수에 입력
   - **주의**: `@` 기호를 포함한 전체 문자열을 복사하세요

---

## 방법 2: 카카오톡 비즈니스 센터에서 확인

### 단계별 가이드

1. **카카오톡 비즈니스 센터 접속**
   - [https://business.kakao.com](https://business.kakao.com) 접속
   - 로그인 (카카오 계정 사용)

2. **채널 선택**
   - 연동하려는 비즈니스 채널 선택
   - 여러 채널이 있는 경우 해당 채널 클릭

3. **채널 설정 페이지 이동**
   - 좌측 메뉴에서 **"채널 설정"** 또는 **"기본 설정"** 클릭

4. **플러스친구 정보 확인**
   - 채널 기본 정보 섹션에서 확인
   - **"플러스친구 ID"** 또는 **"채널 ID"** 항목 찾기
   - 형식: `@보듬교육` 형태

5. **복사**
   - 플러스친구 ID를 복사

---

## 플러스친구 ID 형식

- **형식**: `@` 기호로 시작하는 문자열
- **예시**: 
  - `@보듬교육`
  - `@mybusiness`
  - `@채널이름`

## Vercel 환경 변수에 설정하기

1. [Vercel Dashboard](https://vercel.com/dashboard) 접속
2. 프로젝트 선택 → **Settings** → **Environment Variables**
3. **Add New** 클릭
4. 다음 정보 입력:
   - **Name**: `SOLAPI_PF_ID`
   - **Value**: 복사한 플러스친구 ID (예: `@보듬교육`)
   - **Environment**: Production, Preview, Development 모두 체크
5. **Save** 클릭
6. **재배포 필수**: Deployments → 최신 배포 → ... → Redeploy

## 문제 해결

### 플러스친구 ID가 보이지 않는 경우

1. **카카오톡 비즈니스 채널 연동 확인**
   - 솔라피에서 카카오톡 비즈니스 채널이 제대로 연동되었는지 확인
   - 솔라피 대시보드 → 카카오톡 → 채널 관리에서 채널 상태 확인

2. **플러스친구 검수 완료 확인**
   - 플러스친구가 검수를 통과했는지 확인
   - 검수가 완료되지 않으면 알림톡 발송이 불가능합니다

3. **솔라피 카카오톡 서비스 활성화 확인**
   - 솔라피에서 카카오톡 서비스가 활성화되어 있는지 확인
   - 필요시 솔라피 고객센터에 문의

### ID 형식이 다른 경우

- 일반적으로 `@` 기호가 포함된 전체 문자열을 사용합니다
- 일부 경우 `@` 없이 ID만 표시될 수 있으나, 보통은 `@` 포함 전체를 사용합니다
- 솔라피에서 복사한 값을 그대로 사용하는 것을 권장합니다
