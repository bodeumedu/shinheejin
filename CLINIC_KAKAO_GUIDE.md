# 클리닉 대장 카카오톡 알림톡 템플릿 가이드

## 1. 기능 개요

클리닉 대장 페이지에서 입력된 모든 정보를 카카오톡 알림톡으로 발송할 수 있습니다.

## 2. 포함되는 정보

클리닉 대장의 다음 정보들이 모두 포함됩니다:

- **주차 정보**: 선택된 주차 (예: "2024년 45주차 (11/4~)")
- **요일별 정렬**: 월요일부터 일요일 순서로 정렬
- **시간별 정렬**: 각 요일 내에서 시간 순서로 정렬
- **학생 정보**:
  - 이름
  - 학년 (약자 포함: 천=과천고등학교, 앙=과천중앙고등학교)
  - 반
  - 클리닉 시간 (요일, 시간)
  - 출결 상태 (O/X/결석)
  - 담당 조교님
  - 등원/하원 시간
  - 문자 완료 상태
  - 시험 확인 상태
  - 비고

## 3. 카카오톡 템플릿 등록

### 3.1 템플릿 내용

카카오톡 비즈니스 센터에서 다음 내용으로 템플릿을 만드세요:

```
📋 클리닉 대장 안내

${week}

━━━━━━━━━━━━━━━━━━━━

${content}

━━━━━━━━━━━━━━━━━━━━

보듬교육
```

### 3.2 템플릿 변수

- **${week}**: 주차 정보 (예: "2024년 45주차 (11/4~)")
- **${content}**: 클리닉 대장 전체 내용 (자동 생성)

### 3.3 템플릿 검수 신청 시 샘플 값

검수 신청 시 다음 샘플 값을 사용하세요:

**변수 값**:

  "week": "2024년 45주차 (11/4~)",

## 4. 사용 방법

1. 클리닉 대장 페이지에서 원하는 주차 선택
2. **"📱 카카오톡 전송"** 버튼 클릭
3. 전화번호 입력 (예: 01012345678)
4. 템플릿 코드 입력 (솔라피에서 복사한 템플릿 코드)
5. 자동으로 클리닉 대장 내용이 포맷팅되어 발송됨

## 6. 주의사항

1. **템플릿 코드**: 매번 발송 시 템플릿 코드를 입력해야 합니다. (향후 개선 예정)
2. **내용 길이**: 클리닉 대장 내용이 많을 경우 메시지가 길어질 수 있습니다.
3. **빈 필드**: 입력되지 않은 필드는 자동으로 생략됩니다.
4. **주차 선택**: 현재 선택된 주차의 데이터만 발송됩니다.

## 7. 템플릿 코드 확인 방법

1. [솔라피](https://solapi.com) 로그인
2. 대시보드 → **메시지** → **알림톡** (또는 **카카오톡**) → **템플릿 목록**
3. 승인된 템플릿의 **템플릿 코드** 복사 (예: `TL-1234567890`)

**참고**: 카카오톡 비즈니스 센터에서 검수가 승인되어야 솔라피에서도 템플릿이 보입니다.

## 8. 문제 해결

### 템플릿 코드를 찾을 수 없을 때
- 솔라피 대시보드에서 템플릿이 승인되었는지 확인
- 카카오톡 비즈니스 채널이 솔라피에 연동되었는지 확인

### 메시지가 발송되지 않을 때
- 전화번호 형식 확인 (01012345678)
- 템플릿 코드가 정확한지 확인
- 솔라피 API 키 설정 확인

### 내용이 누락될 때
- 클리닉 대장에 데이터가 입력되어 있는지 확인
- 주차 선택이 올바른지 확인
