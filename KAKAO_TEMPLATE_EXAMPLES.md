## 3. 템플릿 등록 방법

### 3.1 카카오톡 비즈니스 센터 접속

1. [카카오톡 비즈니스 센터](https://business.kakao.com) 접속 및 로그인
2. **플러스친구 선택**: 관리 중인 플러스친구 클릭
3. **비즈메시지 → 알림톡** 메뉴 클릭
4. **템플릿 만들기** 버튼 클릭

### 3.2 템플릿 작성

1. **템플릿 유형**: 알림톡 선택
2. **템플릿 이름**: 
   - "과제 진행 상황 안내" (또는 원하는 이름)
   - "클리닉 대장 안내" (또는 원하는 이름)
3. **메시지 내용**: 위의 템플릿 내용을 그대로 복사하여 붙여넣기
4. **변수 확인**: `#{과제제목}`, `#{과제내용}` 또는 `#{주차정보}`, `#{클리닉내용}` 형식이 올바른지 확인

### 3.3 저장 및 검수 신청

1. **저장** 버튼 클릭
2. 템플릿 목록으로 돌아가기
3. 저장한 템플릿 클릭
4. **검수 신청** 버튼 클릭 (중요!)
5. 검수 승인 대기 (1-3일)

### 3.4 검수 승인 후

1. 템플릿 상태가 "승인"으로 변경되면 사용 가능
2. 솔라피 대시보드에서 템플릿 코드 확인
3. 템플릿 코드를 복사하여 시스템에서 사용

## 4. 주의사항

### 변수 이름
- 변수는 반드시 `#{변수명}` 형식으로 작성
- 영문자, 한글, 숫자, -, _ 20자 이내 사용 가능
- 예: `#{과제제목}`, `#{과제내용}`, `#{주차정보}`, `#{클리닉내용}` (O)
- 예: `${변수명}`, `#{변수명}`, `#변수명` (X)
- **중요**: 변수명은 한글을 사용할 수 있으며, 의미를 명확히 하는 것이 좋습니다

### 템플릿 길이
- 메시지 본문 최대 1000자 이내
- 실제 변수 값이 들어가도 1000자 이내여야 함
- 클리닉 대장의 경우 학생이 많으면 내용이 길어질 수 있음

### 검수 시 주의
- 실제 변수 값이 포함된 샘플 메시지를 작성하여 검수 신청
- 변수명이 정확히 일치하는지 확인
- 이모지 사용 가능하지만 과도한 사용은 피하는 것이 좋음

### 템플릿 코드
- 검수 승인 후 솔라피에서 확인 가능
- 템플릿 코드는 예: `TL-1234567890` 형식
- 시스템에서 템플릿 코드 입력 시 사용

## 5. 템플릿별 사용 방법

### 과제 진행 상황 템플릿
1. 과제 진행 상황 페이지에서 "카카오톡 전송" 버튼 클릭
2. 전화번호 입력
3. 템플릿 코드 입력 (과제 진행 상황 템플릿의 코드)
4. 자동으로 학생별 진행 상황이 포맷팅되어 발송

### 클리닉 대장 템플릿
1. 클리닉 대장 페이지에서 원하는 주차 선택
2. "📱 카카오톡 전송" 버튼 클릭
3. 전화번호 입력
4. 템플릿 코드 입력 (클리닉 대장 템플릿의 코드)
5. 자동으로 주차별 클리닉 대장 내용이 포맷팅되어 발송

## 6. 문제 해결

### 템플릿이 보이지 않을 때
- 플러스친구 검수가 완료되었는지 확인
- 비즈메시지 메뉴가 활성화되어 있는지 확인

### 검수 신청 버튼이 없을 때
- 템플릿 상태가 "임시저장"인지 확인
- 템플릿 상세 페이지에서 검수 신청 버튼 위치 확인

### 솔라피에서 템플릿이 안 보일 때
- 카카오톡에서 검수 승인 완료 여부 확인
- 솔라피에서 카카오톡 채널 연동 상태 확인
- 템플릿 동기화 시간 대기 (승인 후 몇 시간 소요 가능)

### 변수가 제대로 안 들어갈 때
- 템플릿의 변수명과 코드의 변수명이 정확히 일치하는지 확인
- 대소문자 구분 확인 (`${title}` ≠ `${Title}`)
