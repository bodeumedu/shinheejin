# 영어 지문 정리 프로그램

영어 지문을 입력하면 한 페이지에 주제, 요약, 도식화, 그림 등을 깔끔하게 정리해주는 웹 애플리케이션입니다.

## 주요 기능

- 📝 영어 지문 입력 및 정리
- 📌 주제, 요약, 도식화, 그림 영역 구성
- 📱 모바일 반응형 디자인
- 🖨️ 인쇄 최적화 스타일
- 🎨 학생들이 보기 좋은 깔끔한 UI

## 설치 및 실행

### 필요한 도구
- Node.js (v16 이상)
- npm 또는 yarn

### 설치
```bash

npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요.

### 빌드
```bash
npm run build
```

## 사용 방법

1. **OpenAI API 키 설정**
   - 처음 실행 시 API 키 입력 화면이 표시됩니다
   - [OpenAI API 키 발급 페이지](https://platform.openai.com/api-keys)에서 API 키를 발급받으세요
   - API 키는 브라우저 로컬 스토리지에 저장되며, 서버로 전송되지 않습니다

2. **영어 지문 입력**
   - 영어 지문을 입력란에 붙여넣습니다

3. **자동 분석**
   - "정리하기" 버튼을 클릭하면 AI가 자동으로 분석합니다
   - 다음 8가지 항목이 자동 생성됩니다:
     - 한글 주제문
     - 영어 제목
     - 주제가 가장 잘 보이는 문장
     - 영어 요약문 (지문 내 단어 이용)
     - 영어 요약문 (지문 이외 단어 이용)
     - 한글 도식화
     - 영어 도식화
     - 4컷 만화 (내용을 4부분으로 분할)

4. **인쇄 및 저장**
   - 정리된 페이지를 인쇄하거나 PDF로 저장할 수 있습니다
