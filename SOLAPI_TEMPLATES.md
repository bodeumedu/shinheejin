# 솔라피 카카오톡 알림톡 템플릿 검수용 안내

앱에서 사용하는 알림톡 템플릿별 **변수명**과 **템플릿 문구 예시**입니다. 솔라피(카카오 비즈메시지)에 템플릿 등록·검수 시 아래 내용을 그대로 사용하면 됩니다.

---

## 1. 학생 데이터 개별 안내 (학생 데이터 화면 → 행별 카톡 보내기)

**용도:** 학생 데이터 모달에서 메시지 입력 후 해당 학생 행의 「카톡 보내기」로 개별 발송할 때 사용합니다. 입력한 메시지는 `#{공지}` 에 들어갑니다.

**템플릿 코드 (검수 후 발급된 코드로 교체):**  
현재 앱에는 `KA01TP_STUDENT_DATA_INDIVIDUAL` 플레이스홀더가 들어가 있습니다. 솔라피에서 아래 문구로 신규 템플릿 등록 후 발급된 코드를 `StudentDataModal.jsx` 상단 `STUDENT_DATA_KAKAO_TEMPLATE` 에 넣어주세요.

**변수:**

| 변수명     | 설명           | 예시        |
|------------|----------------|-------------|
| #{학생명}  | 학생 이름      | 홍길동      |
| #{학년}    | 학년           | 고2         |
| #{반명}    | 반 이름        | 고2 통합영어 |
| #{공지}    | 발송할 메시지 (메시지 입력란 내용) | 오늘 수업 안내... |

**템플릿 문구 예시 (솔라피 검수 등록용):**

```
#{학생명} (#{학년} / #{반명})

#{공지}
```

---

## 2. 숙제·과제 완료도 전송 (숙제 과제 완료도 화면)

**용도:** 완료도 전송(개별/단체), 날짜별 완료도 전송 시 사용합니다.

**현재 템플릿 코드:** `KA01TP260119030638192BnlwNmKPy78`

**변수:**

| 변수명         | 설명                 | 예시                    |
|----------------|----------------------|-------------------------|
| #{학생명}      | 학생 이름            | 홍길동                  |
| #{과제목록}    | 과제 목록 (줄바꿈)   | 수학 1-10\n영어 단어    |
| #{과제완료상태} | 완료/미완료+메모     | 수학: 완료 (p.5)\n...   |
| #{진도상황}    | 진도 항목별 입력값   | 동사의모든것: p.10\n... |
| #{학년}        | 학년                 | 고2                     |
| #{반명}        | 반 이름              | 고2 통합영어 (월금 14:30) |

**템플릿 문구 예시 (솔라피 등록용):**

```
[보듬교육] #{학생명}님 과제 완료도 안내

#{과제목록}

━━━━━━━━━━━━━━━
#{과제완료상태}
━━━━━━━━━━━━━━━

■ 진도
#{진도상황}

#{학년} #{반명}
감사합니다. 보듬교육
```

**현재 템플릿 코드:** `KA01TP251128032935416MaQYKznwmMv`

**변수:**

| 변수명       | 설명           | 예시        |
|-------------|----------------|-------------|
| #{학생명}   | 학생 이름      | 홍길동      |
| #{반}       | 반 이름        | 고2 통합영어 |
| #{학년}     | 학년           | 고2         |
| #{클리닉날짜} | 주차/날짜 정보 | 1주차 1/6~1/12 |

## 학생 데이터 개별 템플릿 코드 적용 방법

1. 솔라피(카카오 비즈메시지)에서 위 **1번** 문구와 변수(학생명, 학년, 반명, 공지)로 템플릿을 등록·검수받습니다.
2. 검수 통과 후 발급된 **템플릿 코드**를 복사합니다.
3. 프로젝트에서 `src/components/StudentDataModal.jsx` 를 열고, 상단 다음 부분을 수정합니다.

```javascript
const STUDENT_DATA_KAKAO_TEMPLATE = 'KA01TP_STUDENT_DATA_INDIVIDUAL';
```

위 `'KA01TP_STUDENT_DATA_INDIVIDUAL'` 를 발급받은 코드(예: `KA01TP2603XXXXXXXXXX`)로 바꾼 뒤 저장하고 다시 배포하면, 학생 데이터에서 개별 카톡 보내기가 해당 템플릿으로 동작합니다.
